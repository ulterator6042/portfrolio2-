# Portfolio Website

This is a modern portfolio website with three sections:
- **Homepage**: Welcome and download button for portfolio (placeholder).
- **About Me**: Digital CV and contact info (placeholder).
- **Play**: Gallery for hobbies and projects (placeholder images).

## Features
- Full-screen static pages, no scrolling.
- Top floating navigation bar with sliding transitions.
- Glassmorphism UI style.
- Day/Night mode toggle.

## How to Use
Open `src/index.html` in your browser. Replace placeholder text and images with your own content as needed.
